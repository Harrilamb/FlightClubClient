You're about to start the Flight Club tutorial! Ready to become a rocket scientist?

The menu in the top-right corner contains a bunch of pre-built missions from the past and some selected future missions too. You can run these as they are, or you can use them as templates for building your own missions. For example, if you want to build a mission using Falcon 9, it would make a lot of sense to take an existing Falcon 9 mission profile and you should only have to make some small modifications.

Open the menu and choose the mission 'Iridium Next Mission 1'

Great job! Now you'll notice that the list of Events has been populated. This is the stock flight profile for Iridium Next Mission 1. If you wanted, you could hit "Run Simulation!" now and you'd get a lovely result which is my closest guess for this mission's profile. But we're gonna mess with it a bit first.

See those 3 tabs at the top of the screen, just under the toolbar? To choose a launch site for your mission, go to the first tab (yep, the one that says 'LAUNCH SITE'. Good guess.). This part is pretty easy: click the one you want. If it turns gold, it's been selected. Did you really need a walkthrough for that?

Go do it anyway! Select Boca Chica, Texas.

Ok this is going great so far, but here's where it starts to get hard. The 2nd of those 3 tabs (the one that says 'LAUNCH VEHICLE') is where you rocket scientists are gonna go to build your monstrosities. Go there now!

Choose "Use vehicle from existing mission" and you will get access to the dropdown menu beside it, where you can select a mission to copy a vehicle from. Since we selected the Iridium mission in Step 1, that mission's vehicle is chosen here by default.

Click "Customize" and you will be able to mess around with the vehicle structure itself - you can add more stages using this big '+' button (if you don't want a stage, click the 'X' on the stage to be removed), add boosters to stages by clicking the '+' on the stage, and mess with the vehicle and engine specifications by using the edit button in between the '+' and the 'X'!

Choose "Customize" and then click the edit button on the first stage: 'Falcon 9 FT Stage 1'.

Just like with Missions, you can choose a pre-built Stage template here to get you started

Then you can add and edit all of the Engines to be attached to the Stage here!

Finally, you can customize all other (non-Engine) properties of the Stage here, like dry mass, propellant mass, fairing properties, etc.

Play around for a while! When you're ready to continue the tutorial, come to the "FLIGHT PROFILE" tab.

This is the most important part of the whole site: creating a flight profile. If you've ever watched a rocket launch, you know what the flight profile should look like already: 1st Stage Ignition followed by Launch. Some Guidance before 1st Stage Cutoff and Stage Separation, then the 2nd Stage Ignition. Some more Guidance maneuvers for a few minutes before the 2nd Stage Cutoff.

Done. The semantics are just "What time does each thing happen?" and "What parameters should the Guidance maneuvers have?" That's really all there is to it.

For example, click "Edit Event" on the first Event in the list: "Main Engine Ignition".

Here we have 2 fields where you can set the time the event is due to happen, and you can also set a name for the Event. The name is for display purposes only, so be creative and descriptive!

For Guidance type events, you will also see a field here marked "End". Guidance maneuvers are not meant to be instantaneous, so a start and an end time is required. For example, a throttle maneuver with a throttle setting of 0.5, a start time of 0, and an end time of 10 will steadily increase or decrease the stage's throttle so that it reaches 0.5 at exactly T+10. Leaving the "End" field empty will cause the stage to perform the maneuver as fast as it can.

The Event Type dropdown gives you the different options for Events, including Ignition, Cutoff, Stage Separation, Fairing Separation and more. Your choice here will change the availability of some other fields, but we'll come back to that.

The Stage field is obvious: which Stage does this Event apply to? You can choose multiple Stages for one Event.

For Ignition and Guidance type events, you can add Heading and Throttle values. Pitch is defined as degrees above or below level flight. So an airplane (should) fly at 0°, while a rocket on the pad has a pitch of 90°. This property must obey -90°<Pitch<90°

Yaw is defined as degrees counterclockwise of Eastward flight. 90° is flying towards the North Pole, 180° is flying West, and 270° is flying South.

Throttle takes a floating point number and must obey 0<Throttle<1.

Finally, Gravity Turn can be used as a shortcut in certain situations if it's difficult or impossible to know what Pitch and Yaw should be. A Forward Gravity Turn aligns the Thrust vector with the velocity vector (so you're thrusting in the direction of motion) and a Reverse Gravity Turn does exactly the opposite. Reverse is very useful for retropropulsion.

For Ignition and Cutoff type events, you can set the number of engines to be Ignited/Cutoff. For example for SpaceX's landing burns, not all engines are lit. In vehicles with more than one engine type, there will be a separate input field for each engine here.

If you want to land your rocket stage safely on the ground again, it can be tricky to get the timing of the landing burn ignition - as well as the throttle settings - exactly right. So Flight Club does it for you! If you want your stage to hit 0m/s velocity at 0m altitude, flick the Dynamic Burn switch. Use with caution though, if activated too far above the surface, these calculations can clog up your entire simulation. Try and wait until you have around 5-10km altitude left before starting these burns.
